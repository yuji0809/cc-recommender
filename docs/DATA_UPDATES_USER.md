# Data Updates

## 更新頻度

cc-recommenderのデータベースは**毎週月曜日 AM 9:00 JST**に自動更新されます。

### 更新内容

- 新しいプラグイン（公式マーケットプレイス）
- 新しいMCPサーバー（awesome-mcp-servers）
- 新しいスキル/ワークフロー（awesome-claude-code）
- セキュリティスコアの再計算

## 自動更新機能

### 仕組み

1. **サーバー起動時に最新データを取得**
   - GitHubから最新の`recommendations.json`を自動取得
   - 3秒以内にレスポンスがない場合はバンドル版にフォールバック

2. **手動更新不要**
   - npmパッケージを再インストールする必要なし
   - 常に最新のデータで推薦を受けられます

3. **安全性**
   - データはバリデーション済み
   - HTTPS通信のみ
   - 取得失敗時はバンドル版データを使用

### オフラインモード

ネットワークアクセスを無効にしたい場合:

```json
{
  "mcpServers": {
    "cc-recommender": {
      "command": "npx",
      "args": ["-y", "cc-recommender"],
      "env": {
        "OFFLINE_MODE": "true"
      }
    }
  }
}
```

オフラインモードでは、バンドルされたデータのみを使用します。

## データの信頼性

- ✅ すべてのアイテムは週次でセキュリティスキャン済み
- ✅ セキュリティスコアが付与されています
- ✅ 公式ソースから取得

## よくある質問

### Q: どのくらいの頻度で更新されますか？
A: 毎週月曜日に自動更新されます。

### Q: 手動で更新する必要がありますか？
A: いいえ、サーバー起動時に自動的に最新データを取得します。

### Q: オフラインでも動作しますか？
A: はい、リモートデータの取得に失敗した場合は、バンドルされたデータを使用します。

### Q: データの取得に時間がかかりますか？
A: 通常は1秒以内です。3秒でタイムアウトし、バンドル版にフォールバックします。

### Q: どのデータソースから取得していますか？
A: 公式のGitHubリポジトリから取得しています:
- anthropics/claude-plugins-official
- punkpeye/awesome-mcp-servers
- hesreallyhim/awesome-claude-code
